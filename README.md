# aaaa-gcw0

Controls:

A: Confirm/Sword

B: Jump

X: Shout/Confirm Quit

Y: Fire Gun

L: Aim gun

R: Look

Start Enter/Menu

Select Esc/Torch

DPAD: Movement

Joypad: Analogue movement

HOLD: Exit to OS immediately


Known bugs:

Textures flicker with a certain camera orientation.

Narrator voice missing in into sequence.

G-Sensor is buggy.
