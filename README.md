# aaaa-gcw0

Controls:

A: Confirm/Sword

B: Jump

X: Shout/Confirm Quit

Y: Fire Gun

L: Aim gun

R: Look

Start Enter/Menu

Select Esc/Torch

DPAD: Movement

Joypad: Analogue movement

HOLD: Exit to OS immediately


Known bugs:

Force feedback is glitchy due to GCW0's driver (needs firmware fix)

PC Build joystick axes are wrong
